## 🚀 기능 구현 목록

### 메인 로직

- [ ] 컴퓨터 랜덤 숫자 구현
- [ ] 출력함수 구현(원하는 메시지 출력) - (ex) 종료, 시작 메시지)
- [ ] 입력함수 구현(숫자 입력, 재시작 입력)
- [ ] 컴퓨터 랜덤 값과 입력값 비교 함수
- [ ] 비교 결과값 문자로 변환 함수(ex) 1볼 1스트라이크)

### 에러 관련 로직

- [x] isNumber(): (숫자야구 상황) 입력값이 숫자인지 아닌지 확인
- [x] is3DifferNumber(): (숫자야구 상황): 서로 다른 3자리 수인지 확인하는 함수
- [ ] 재시작/종료 상황: 입력값이 1아니면 2인지 확인하는 함수
